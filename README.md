# PGRC
Il progetto si pone l’obiettivo di sviluppare l’applicazione web Piattaforma per la Gestione di Ricette di Cucina (PGRC). PGRC implementa un portale web per la gestione di ricette culinarie dove un utente registrato pu`o gestire il suo ricettario personale partendo da un vasto insieme di piatti predefiniti.
